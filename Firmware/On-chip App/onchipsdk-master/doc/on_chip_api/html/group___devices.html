Description

Access the mote's devices, including sensors and digital interfaces.

The SmartMesh library gives your application access to the devices present on the LTC5800 mote. The term "devices" is generic and includes:

  • internal sensors such as Battery voltage and Temperature,
  • the built in analog to digital converter (ADC),
  • hardware interfaces to communicate with an external sensors or actuator such as SPI, I2C, UART or 1-Wire,
  • general purpose input/ouput (GPIO) pins to drive an external actuator or wake up on an external event,
  • a command line interface (CLI) serial port to interact with a human,
  • information about the state of the flash memory (flashinfo),
  • a random number generator.
Warning
The device API is not thread-safe. You should only access a particular device from a single thread.

We group these devices into two categories:

  • Sensors
  • Other

Your application interacts will all these devices using the same Device API.
