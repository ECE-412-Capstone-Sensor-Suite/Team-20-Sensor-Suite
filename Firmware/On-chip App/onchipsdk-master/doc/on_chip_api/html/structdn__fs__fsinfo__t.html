Detailed Description

Information about the file system.

Field Documentation

shortPageSize

The number of data bytes that fit into a single short page.

This number might be different form dn_fs_fsinfo_t::longPageSize in the way control information is handled.

Each file is associate some control information:

  • For "short" files, this control information lives in the same flash memory page as the file content. The bytes available for file content is therefore smaller than the size of a page.
  • For "long" files, this control information is stored outside of the file content, leaving the whole flash memory page to file content.
longPageSize

Number of data bytes in long (>1 page) files.

This number might be different form dn_fs_fsinfo_t::shortPageSize in the way control information is handled.

  • For "long" files, this control information is stored outside of the file content, leaving the whole flash memory page to file content.
