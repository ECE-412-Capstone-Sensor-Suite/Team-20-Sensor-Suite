dn_pwn_periods Special Period Values for no-glitch transitions.

The PWM engine can only "smoothly" change the duty cycle while running for
several specific period values.

Other period values can be used, but will result is a single "glitched" cycle which may be either a full period of high duty cycle or a single slighty longer period. The period values that can product no-glitches when changing the duty cycle are listed here.
