Detailed Description

dn_open() argument.

Pass a pointer to a structure of this type when using the dn_open() function with the DN_CLI_DEV_ID device.

Field Documentation

Serial port to use for CLI.

baudRate

Baudrate for the CLI port.

Set this value to 9600, which the only supported baud rate.
