Introduction

This is a companion documentation to the On-Chip Firmware Development Guide.

It documents the Application Programming Interface for programming the LTC5800, i.e. the functions and data structures to use to interact with the SmartMesh library provided as part of the SmartMesh On-Chip Firmware Development Kit.

To build an application on top of the LTC5800, write an application which calls functions defined in this document, compile and link your application against the SmartMesh library, and load the resulting binary onto your LTC5800-based board.

The functions and data structures are grouped by component. Use the "Components" button on the top to get an overview of all components. The organization of the modules mirrors the organization in the On-Chip Firmware Development Guide.

The On-Chip Firmware Development Guide introduces general concepts. This document provides the details of each function and data structure.

Note
Code snippets and sample applications are provided as part of the On-Chip Firmware Development Kit. These snippets can help you understand the API. They are introduced in the On-Chip Firmware Development Guide.
